# Legacy Workspaces

This folder contains archived experiments and side paths that are not part of the active Erasmus Journey MVP runtime.

- `server/`: historical standalone Express + SQLite backend
- `api-service/`: historical NestJS service spike
- `angular-ssr/`: historical Angular SSR experiment

The active product lives at the repository root as a Next.js application. Use the root `README.md` for the current development workflow, and `docs/LEGACY_WORKSPACES.md` for more context on these archived paths.
